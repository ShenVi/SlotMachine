# SlotMachine
基于Python Flask开发的老虎机Dome 禁止商业使用或个人运营

#### 使用方法
下载dome到本地 安装依赖库 启动地址为127.0.0.1 需要mysql支持

在 __config.py__ 内修改数据库地址
